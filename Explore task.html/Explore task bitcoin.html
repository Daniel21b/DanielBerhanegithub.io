Semantics in Web Development

Similarly, semantic elements that make up a web page describe the content being viewed in a meaningful, logical, and sequential way. For example, instead of using nondescript tags such as <div> or <span> to identify an independent element, web developers can use <section> or <article> tags to show how this element functions. Also, the sequence and layout of elements not only determine the site's structure, but also how certain elements inherit traits and behaviors from other elements, depending on their hierarchy in that structure.
